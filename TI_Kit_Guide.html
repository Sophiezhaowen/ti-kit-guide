Getting Started

Step 1: Create a folder on your desktop to store TI Kit files.
Step 2: Check your TI Kit version: we are using MSP-EXP430FR5994 for this class.
Step 3: Download Energia IDE (choose your operating system) and unzip.

Mac users please download from here.

Windows users please download from here here.

  1. Download the zipped file using the link above.
  2. Extract everything to the folder you created in Step 1. Please note that everything MUST be remained in the same folder after extraction.
  3. After extraction, feel free to delete the original zipped folder.
  4. Please check if there is a driver folder in your unzipped folder.

Linux users please download from here here.

Step 4: Configure the course files in Energia.
  1. Open Energia IDE. This will create a folder in your Documents titled “Energia”.
  2. Download the ENGR 131 Sketchbook.
  3. Unzip the ENGR 131 sketchbook file and copy the following folders into the “Energia” folder in your Documents:
    • ENGR 131
    • LaunchPad_Grove_Kit
  4. Close and restart Energia.
  5. Navigate to File → Sketchbook, and you should see the ENGR 131 and LaunchPad_Grove_Kit folders with sketches available to use for class.
Step 5: Setup Energia to communicate with the TI kits.
  1. Connect the MSP430 board to your laptop using a cable.
  2. In Energia, open Tools and set the following:
    • Board → MSP-EXP430FR5994LP
    • Port → Select the port that corresponds to your laptop’s USB/Type-C connection. If you are unsure which port it is, try each option until the board is detected.
